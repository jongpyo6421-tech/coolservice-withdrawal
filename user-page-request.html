사용자 페이지

요청 사항
현재 로그아웃 버튼의 텍스트가 작고 흐려 사용자가 쉽게 인지하기 어려울 것 같습니다.

추가로 위치도 "내 정보 관리" 앞의 아이콘과 정렬 부탁드립니다.
