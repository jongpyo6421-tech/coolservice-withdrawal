요청 사항
우측 상단의 '사용자 차단' 버튼을 제거하고, 아래 두 버튼으로 교체 부탁드립니다.
• 소속 제외 — 해당 사용자의 소속 학교를 초기화하는 기능 (소속이 없는 경우 소속 제외 버튼 하이드)
• 회원 탈퇴 — 해당 사용자의 계정을 탈퇴 처리하는 기능
💡 개선 요청 화면에서 소속 제외·회원 탈퇴 버튼을 클릭하시면 모달을 확인하실 수 있습니다.
